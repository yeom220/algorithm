# Spring starter?
	스프링 초기 프로젝트 생성을 도와주는 서비스. 필요한 라이브러리를 선택하면 빌드 설정 파일에 의존성들을 자동으로 추가하여 프로젝트를 생성 해준다.

# Gradle
	그루비(Grrovy)를 기반으로 한 빌드 자동화, 개발 지원 도구. build.gradle 파일은 빌드 스크립트로 의존성이나 플러그인 설정과 같은 빌드에 필요한 설정을 한다.
